Job Description

Learn to build software from the ground up. Work under a highly experienced team that has built software products and platforms.

We are looking for someone with familiarity of the ASP.NET Core, ASP.NET framework, SQL Server.

  • Knowledge of C#, SQL Server, LINQ, Entity Framework, JavaScript, jQuery.

  • Knowledge of working with ASP.NET Web Forms, Web APIs, .Net Core APIs.
